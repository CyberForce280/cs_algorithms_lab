### أزرار التفاعل

| الزر | الوظيفة |
|------|---------|
| `▶ ابدأ` | تهيئة الخوارزمية بالأرقام المدخلة |

| `↺ إعادة` | إعادة التهيئة من البداية |

### ألوان المرئيات

| اللون | المعنى |
|-------|--------|
| 🟡 أصفر | العنصر الحالي / المحور |
| 🟠 برتقالي | عناصر المقارنة |

| 🟢 أخضر | عناصر في موضعها الصحيح / وُجد |

## 📊 التعقيد الزمني — مقارنة سريعة

| الخوارزمية | أفضل حالة | متوسط | أسوأ حالة |
|------------|-----------|-------|-----------|
| Bubble Sort | O(n) | O(n²) | O(n²) |
| Insertion Sort | O(n) | O(n²) | O(n²) |
| Quick Sort | O(n log n) | O(n log n) | O(n²) |
| Sequential Search | O(1) | O(n) | O(n) |
| Binary Search | O(1) | O(log n) | O(log n) |
